How it works

The subset construction algorithm converts any NFA into an equivalent DFA by treating each set of NFA states as a single DFA state.

  1. Compute the ε-closure of the NFA start state
  2. For each unprocessed DFA state, apply each symbol
  3. Compute ε-closure of the resulting NFA state set
  4. Repeat until no new states are discovered
